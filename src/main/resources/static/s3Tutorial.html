S3 Tutorial


Amazon Simple Storage Service (S3) es un servicio de almacenamiento para Internet. Está diseñado para facilitar a los desarrolladores recursos de computación escalables basados en Web. Los siguientes son los pasos para poder desplegar un sitio estático usando este servicio:

  • Ya con una cuenta en AWS educate, se dirige a la consola de administración de AWS (AWS Management Console).
  • En esa consola se busca el servicio S3 y se accede a él.
  • Dentro de Amazon S3, creamos un bucket para almacenar todos los objetos que se vayan creando.
  • Ahí llenamos los campos que permiten darle un nombre al bucket, se configuran propiedades si se desea, se establecen los permisos necesarios (en este caso, para permitir el despliegue de una pagina estática, es necesario que los objetos sean públicos) y finalmente se crea el bucket.
  • Accedemos al bucket creado y cargamos los documentos de nuestro sitio web estático, como un index.html y error.html. Aquí debemos seleccionar la clase de almacenamiento como standard, pues esto tiene que ver con el costo.
  • Luego de eso seleccionamos nuestros archivos y en actions seleccionamos make public.
  • Ahora vamos a la pestaña de propiedades y hacemos click en static website hosting para poder activar nuestro servicio. Seleccionamos la opción “use this bucket to host a website” y escribimos el nombre de los documentos del index y del error. Finalmente damos click en save.
  • En la parte superior de esa propiedad aparece la dirección que será tendrá el hosting de nuestro sitio estático, lo copiamos y lo abrimos en otra pestaña. Debería mostrarse nuestro index.
